# Daily Reports

AI Platformの設計・実装・運用に関する日次記録を管理します。

## 保存先

```text
Final daily report:
reports/daily/YYYY-MM-DD.md

Work fragments:
reports/daily/fragments/YYYY-MM-DD/NN-topic.md
```

## 運用

- 作業中は主要タスク単位でfragmentを残す
- 作業終了時または重要な区切りでfinal daily reportを整える
- 詳細な設計、コード、Issue、PR、Commitは各正本に残す
- daily reportには要約、判断、確認結果、参照先、次回作業を記録する
- `engineering-career-hq` へは節目だけを同期し、詳細ログを重複させない

詳細は `docs/reporting-policy.md` を参照してください。

## テンプレート

```text
reports/templates/daily-report-template.md
reports/templates/fragment-template.md
```
